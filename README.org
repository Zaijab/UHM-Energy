** Organization
The data can be organized in the following manner.

=Campus= *contains* =Building_Complex= *contains* =Buildings= *contains* =Main Meters= *contains* =Sub Meters= 

*Contains* is a bit of a "catch all" for some kind of encompassing relationship. It can be thought of as the answer to, "What would I need to add up to get a total power reading?". For example, "How much power does UH Manoa draw on January 1st?" would be answered by summing all meters in Manoa. We can answer this equivalently by recursion \( \text{Power}(\text{Manoa}) = \sum_{\text{Building} \in \text{Building Complex}} \text{Power}(\text{Building}) \).

Thinking about the organization of the data in this recursive manner tells us we should use a directed acyclic graph (DAG).
